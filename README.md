# Landing Page — INDEL S.R.L.

Sitio web de INDEL S.R.L. — Equipamiento Industrial (Rosario, Santa Fe).

## Estructura

- `index.html` — página principal (home).
- `producto-template.html` — plantilla reutilizable para páginas de producto.
- `autoelevadores.html`, `puentes-grua.html`, … — páginas de cada producto.
- `img/` — imágenes de productos y marcas.
- `logos-clientes/` — logos de clientes para la home.
- `logo-indel.png.png` — logo de la empresa.

## Cómo trabajar

Es un sitio estático: se abre directamente con doble clic en `index.html`,
no necesita servidor ni instalación.

## Páginas de producto

Para crear una página nueva, copiar `producto-template.html`, renombrarla
(ej.: `aire-comprimido.html`) y reemplazar el contenido marcado con
comentarios `<!-- EDITAR -->`.
